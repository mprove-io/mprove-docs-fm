# About

[Mprove](https://mprove.io/) is an [Open Source](https://github.com/mprove-io/mprove) business intelligence tool.

Mprove's purpose is to be the best way to explore **All Metrics in One Place**:
- SQL data sources, see [Malloy Model](/docs/ref-model-malloy-source).
- HTTP API data sources, see [Store Model](/docs/ref-model-store).

All models, charts, reports and dashboards are stored under git version control. 

## My Story

Hey! I'm Alexander, Mprove CEO, founder and lead developer.

I started working on Mprove in 2015 inspired by Looker. 

Understanding how things should work and look comes through a lot of trial and error. This is my third big attempt to build it right.

Mprove originally had its own language similar to LookML. In 2025, I switched to Malloy as the next generation data modeling language.
