{/* ## Access Roles */}

Access roles can be assigned to project members by **projectAdmin**.

Dashboards, Charts, Reports and Models (we'll refer to them as the objects) have `access_roles` parameter.

If object has `access_roles: [sales, marketing]`, then only those project members who have the "sales" or "marketing" access role will be able to access this object.

If an object has no `access_roles` then any project member can access this object. This means that the object is public for all project members.

<Callout>
  If the user does not have access to a specific Model - he will be able to see
  it in the lists, but will not be able to explore it.
</Callout>

<Callout>
  If the user does not have access to a specific Report, Chart or Dashboard - he
  will not be able to see it in the lists and will not be able to explore it
</Callout>
