# Validate

## Usage

```
➜  ~ mprove validate -h
Validate (rebuild) Mprove Files for selected env

━━━ Usage ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

$ mprove validate

━━━ Options ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  --project-id #0       (required) Project Id
  --repo-type #0        (required, "dev", "production" or "session")
  --branch #0           (required) Git Branch
  --env #0              (required) Environment
  --json                (default false)
  --env-file-path #0    (optional) Path to ".env" file

━━━ Examples ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Validate Mprove Files for Dev repo, env prod
  $ mprove validate --project-id DXYE72ODCP5LWPWH2EXQ --repo-type dev --branch main --env prod

Validate Mprove Files for Production repo, env prod
  $ mprove validate --project-id DXYE72ODCP5LWPWH2EXQ --repo-type production --branch main --env prod
```
