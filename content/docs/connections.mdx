# Connections

## SQL Data Sources

Mprove connects to SQL databases that are supported by Malloy:

- PostgreSQL
- MySQL
- Presto
- Trino
- SnowFlake
- BigQuery
- MotherDuck (DuckDb)

## API Data Sources

The API connection is universal and can work with different API providers.
Logic for creating a request and transforming a response must be defined in the Store file.

Store connection types:

- API
- Google API

For **API** query, Mprove does a single request to the data source.

For **Google API** query, Mprove does 2 requests:

- Auth request to get credentials
- Data request (query)

## API Store Presets

[Store files](/docs/ref-model-store) can work without presets.

Presets are reusable Store file templates for API data sources.

Mprove has a [Store preset for Google Analytics](https://github.com/mprove-io/mprove/blob/master/apps/blockml/src/presets/google_analytics.store.preset).

Minimal Store file that uses the preset looks like this.

<DynamicCodeBlock
  lang="yaml"
  code={googleAnalyticsYaml}
  options={{
    themes: {
      light: "catppuccin-latte",
      dark: "rose-pine",
    },
  }}
/>

Additional parameters specified in the Store file will override those taken from the preset.
