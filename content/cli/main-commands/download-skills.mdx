# Download Skills

## Usage

```
➜  ~ mprove download-skills -h
Download mprove skills to a local directory

━━━ Usage ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

$ mprove download-skills

━━━ Options ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  --output #0           (required) Output directory path
  --env-file-path #0    (optional) Path to ".env" file

━━━ Examples ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Download skills to a directory
  $ mprove download-skills --output ./skills
```

## Example Output

```
➜  ~ mprove download-skills \
--env-file-path /mprove/.env \
--output skills

```
