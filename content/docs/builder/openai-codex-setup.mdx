# OpenAI Codex Setup

You can use OPENAI_API_KEY or Codex Subscription for AI Sessions.

## Codex Subscription
Run `opencode auth login` on your local machine to create opencode's auth.json file.

Run `mprove set-codex-auth` to copy auth.json from your local machine to your mprove server user.

Your AI Session will be able to use Codex Subscription. 

Other mprove users will need their own Codex Subsctiptions.
