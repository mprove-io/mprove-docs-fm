# Install CLI

Mprove CLI is packaged with `NodeJS`. `Git` is a required dependency.

## Using executable binaries (MacOS, Linux)

https://github.com/mprove-io/mprove-cli/releases

## Using package managers

### Homebrew (MacOS, Linux)

Install

```bash
brew install mprove-io/mprove/mprove-cli
```

Update

```bash
brew upgrade mprove-io/mprove/mprove-cli
```
