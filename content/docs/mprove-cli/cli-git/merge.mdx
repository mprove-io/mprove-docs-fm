# Merge

## Usage

```
➜  ~ mprove merge -h
Merge their-branch to branch for Dev repo, validate Mprove Files for selected env

━━━ Usage ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

$ mprove merge

━━━ Options ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  --project-id #0          (required) Project Id
  --branch #0              (required) Git Branch
  --their-branch #0        (required) Their git Branch
  --env #0                 (required) Environment
  --their-branch-remote    (default false), merge from production (remote) repo branch
  --get-errors             (default false), show validation errors in output
  --get-repo               (default false), show repo in output
  --json                   (default false)
  --env-file-path #0       (optional) Path to ".env" file

━━━ Examples ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Merge their-branch to branch for Dev repo, validate Mprove Files for env prod
  $ mprove merge --project-id DXYE72ODCP5LWPWH2EXQ --their-branch b1 --branch main --env prod
```
