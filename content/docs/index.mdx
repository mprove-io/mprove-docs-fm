# About

[Mprove](https://mprove.io/) is a business intelligence tool. [Open Source](https://github.com/mprove-io/mprove).

Mprove's purpose is to be the best way to see "**All metrics in one place**".

Mprove relies on [Malloy](https://www.malloydata.dev/) for SQL data sources, see [Malloy Model](/docs/ref-model-malloy-source).

Mprove can make queries to HTTP API data sources, see [Store Model](/docs/ref-model-store).

Mprove supports git version control. All models, charts, reports and dashboards are stored as files.
