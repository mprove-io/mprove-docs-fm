# Builder

Builder is a lightweight IDE section of Mprove UI.

Users with File Editor role can browse, edit and validate mprove and malloy files.

There are two types of Builder AI sessions:

- Editor Session
- Explorer Session (Under Development)

[Editor Session](/docs/builder/editor-session) can build mprove models and create non-draft reports, charts and dashboards.

Explorer can answer user questions by creating draft charts, reports and dashboards.
