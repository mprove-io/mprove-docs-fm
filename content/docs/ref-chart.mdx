# Chart

Each chart must be stored as **chart_name.chart** YAML file:

<DynamicCodeBlock
  lang="yaml"
  code={chartYaml}
  options={{
    themes: {
      light: "catppuccin-latte",
      dark: "rose-pine",
    },
  }}
/>

## Reference

### Chart

| Name         | Type                                | Default | Description                                                              |
| ------------ | ----------------------------------- | ------- | ------------------------------------------------------------------------ |
| chart\*      | string                              | -       | Chart name                                                               |
| access_roles | string []                           | -       | If specified, only users with the listed roles will have access to Chart |
| tiles\*      | [Tile []](/docs/ref-dashboard#tile) | -       | Chart tile has the same configuration as dashboard tile                  |
