### Usage


```
➜  ~ mprove create-branch -h
Create branch

━━━ Usage ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

$ mprove create-branch

━━━ Options ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  --project-id #0       (required) Project Id
  --repo #0             (required, "dev" or "production")
  --new-branch #0       (required) New Branch name
  --from-branch #0      (required) From Branch name
  --json                (default false)
  --env-file-path #0    (optional) Path to ".env" file

━━━ Examples ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Create branch for Dev repo
  $ mprove create-branch --project-id DXYE72ODCP5LWPWH2EXQ --repo dev --new-branch b1 --from-branch main

Create branch for Production repo
  $ mprove create-branch --project-id DXYE72ODCP5LWPWH2EXQ --repo production --new-branch b1 --from-branch main
```


### Example Output


```
mprove create-branch \
--env-file-path ~/mprove/.env \
--project-id DXYE72ODCP5LWPWH2EXQ \
--repo dev \
--new-branch b2 \
--from-branch main

message: Created branch "b2"
```
