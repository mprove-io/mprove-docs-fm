# Chart

Each chart must be stored as **chart_name.chart** YAML file:

```yml lineNumbers
chart: chart_name
   
tiles: # Chart must have exactly one tile
# - title: 'tile title'
#   ...

access_roles:
- role
- role
```

## Reference

### Chart

| Name         | Type                                      | Default | Description                                                              |
| ------------ | ----------------------------------------- | ------- | ------------------------------------------------------------------------ |
| chart\*      | string                                    | -       | Chart name                                                               |
| tiles\*      | [Tile](/top/reference/dashboard#tile) [ ] | -       | Chart tile has the same configuration as dashboard tile                  |
| access_roles | string [ ]                                | -       | If specified, only users with the listed roles will have access to Chart |
