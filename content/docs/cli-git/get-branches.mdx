# Get Branches

## Usage

```
➜  ~ mprove get-branches -h
Get branches

━━━ Usage ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

$ mprove get-branches

━━━ Options ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  --project-id #0       (required) Project Id
  --repo-type #0        (required, "dev", "production" or "session")
  --json                (default false)
  --env-file-path #0    (optional) Path to ".env" file

━━━ Examples ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Get Dev repo branches
  $ mprove get-branches --project-id DXYE72ODCP5LWPWH2EXQ --repo-type dev

Get Production repo branches
  $ mprove get-branches --project-id DXYE72ODCP5LWPWH2EXQ --repo-type production
```

## Example Output

```
➜  ~ mprove get-branches \
--env-file-path /mprove/.env \
--project-id DXYE72ODCP5LWPWH2EXQ \
--repo-type dev

branches: 
  - main
```
